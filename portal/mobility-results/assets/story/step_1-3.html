Für das erste Realexperiment wurden zwei digitale Tools entwickelt: Eines zur Erhebung von Mobilitätsdaten und eines zum Storytelling. Im Anschluss wurden die Tools von rund 20 unbezahlten Sorgearbeiter:innen getestet. Die vorläufigen Erkenntnisse wurden bei einer Abendveranstaltung öffentlich präsentiert und diskutiert. Die erstellte Daten-Story über das Mobilitätsverhalten von unbezahlten Sorgearbeiter*innen sowie das Tools zur Datenerhebung können Sie unter diesem Link aufrufen.
