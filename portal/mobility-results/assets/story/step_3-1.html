Sollten Sie Interesse an der Weiternutzung der Codes zum Storytelling (Urban Data Narrator) oder zur Datenerhebung (Urban Data Collector) haben, können Sie diese auf der GitHub Seite des City Science Labs finden github.com/citysciencelab. Die Tools sind open source und als Addons für das Masterportal angelegt. Eine ausführliche Anleitung zur Nutzung finden sie ebenfalls in den besagten Repositories. Entwickelt wurde die Software gemeinsam mit Ubilabs.
